# BookKeeping-for-Corona-Patients-CommandLine-App
(Advanced Programming 2020 Refresher Assignment)


It is a bookKeeping GUI app that uses Swing and Java Data Structures to keep track of the date of infection and recovery time of all residents in 4 towers A,B,C and D and according allot them hospital according to the number of beds availaible in each hospital and the patient's condition.
